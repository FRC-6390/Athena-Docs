# Install Hugo
https://gohugo.io/getting-started/installing/

# Install Learn Theme

```
cd themes/
git clone https://github.com/matcornic/hugo-theme-learn.git
```

# Run server

```
hugo server
```

# Create Pages

## Chapters
```
hugo new chapter 'content\<folder>\_index.md'
```
## Pages
```
hugo new page 'content\<folder>\<name>.md'
```
